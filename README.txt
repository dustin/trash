NAME
     trash — A safe commandline alternative to rm.

SYNOPSIS
     trash [file ...]

DESCRIPTION
     trash This is a simple commandline utility to throw things you don’t want
     into your workspace trash.

     Note that in order to use this, you must have a session logged in as it
     uses NSWorkspace to accomplish its move.
